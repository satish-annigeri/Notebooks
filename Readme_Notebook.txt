Steps to open IPython Notebooks
-------------------------------
1. Open MS-DOS prompt
2. Change over to the folder where the Notebook files are stored, using cd command at MS-DOS prompt
3. Type the following command at the MS-DOS prompt
   ipython notebook <cr>
4. Your browser will open if it is not yet opened and in a new tab you will see the IPython Notebook dashboard with the list of all Notebooks in the folder from where IPython Notebook server is started
5. Click on any Notebook file to open it. You can edit and save existing Notebook files. To create a new Notebook file, go to the dashboard and click on New Notebook.
